Automatische
Untertitel für
deine Videos.

Untertitel-Programm erkennt gesprochene Sprache in Videos automatisch, lässt dich die Untertitel bearbeiten und brennt sie direkt ins Video ein — alles lokal auf deinem PC.

Windows Kostenlos nutzbar KI-gestützt
So funktioniert es

Von der Aufnahme zum fertigen Video in drei Schritten.

SCHRITT 01

Video hochladen

Lade dein Video in das Programm. Es unterstützt MP4, MOV und weitere Formate.

SCHRITT 02

Untertitel prüfen

Die KI erkennt die gesprochene Sprache. Du kannst Timing und Text manuell anpassen.

SCHRITT 03

Exportieren & posten

Das fertige Video wird gerendert und kann direkt auf TikTok hochgeladen werden.

TikTok Integration

Direkt auf TikTok hochladen.

Untertitel-Programm nutzt die offizielle TikTok API um Videos direkt auf dein Konto hochzuladen — sicher, transparent und vollständig unter deiner Kontrolle.

Authentifizierung über den offiziellen TikTok Login Kit (OAuth 2.0)

Videos werden nur hochgeladen wenn du es aktiv bestätigst oder einen Zeitplan festlegst

Keine Daten werden auf externen Servern gespeichert — nur dein finales Video wird an TikTok übertragen

Zugriff kann jederzeit in deinen TikTok-Einstellungen widerrufen werden

Deine Daten gehören dir.

Untertitel-Programm speichert keine persönlichen Daten auf externen Servern. Alle Verarbeitung findet lokal auf deinem PC statt. Die einzigen Daten die das Programm an externe Dienste sendet sind das fertige Video beim Upload zu TikTok und die für die TikTok-Authentifizierung notwendigen OAuth-Tokens.
